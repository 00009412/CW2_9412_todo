CW2_9412_todo
=============

About
-----

An app to create and organize tasks using lists.

How to run
----------

Assuming you are at the root of the project:

1. Install the dependencies `npm i`
2. Run `npm run watch` to start the app
3. Expect the app to run at `localhost:3000`
